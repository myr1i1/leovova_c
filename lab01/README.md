# Лабораторная работа №1
## Типы данных, память и выражения в С

### Цель работы
Научиться работать с базовыми типами данных языка C, понимать связь переменных и памяти, использовать операции и выражения, выполнять ввод и вывод данных, понимать переполнение и точность вычислений.

### Используемые технологии
- Язык программирования: C
- Компилятор: gcc
- Операционная система: Ubuntu (через VirtualBox)

### Инструкция по запуску
1. Открыть терминал
2. Перейти в папку с заданием: cd ~/lab01
3. Скомпилировать нужный файл: 
   gcc task1.c -o task1
4. Запустить программу:
   ./task1(и так далее до 14го)
