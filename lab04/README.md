# Лабораторная работа №4
## Ввод-вывод и работа с файлами в языке C

### Цель работы
Освоить работу с файлами, открытие и закрытие потоков, чтение и запись текстовых и бинарных данных, форматированный ввод-вывод, перемещение по файлу.

### Задания
1. Открытие и закрытие файла
2. Посимвольная запись и чтение (putc/getc)
3. Работа со строками (fputs/fgets)
4. Форматированный ввод-вывод (fprintf/fscanf)
5. Ввод с клавиатуры и запись в файл
6. Бинарные файлы (fwrite/fread) – массив int
7. Структура Person в бинарном файле
8. Массив структур
9. Позиционирование (fseek)
10. Размер файла (ftell)
11. Проверка конца файла (feof) и ошибок (ferror)
12. Простая файловая база людей


### Используемые технологии
- Язык программирования: C
- ОС: Ubuntu на VirtualBox
- Компилятор: gcc
- Библиотека: stdlib.h

### Инструкция по запуску
1. Открыть терминал
2. Перейти в папку lab03: `cd ~/leonova_c/lab04`
3. Скомпилировать файл: `gcc task1.c -o task1`
4. Запустить: `./task1`

### Ответы на вопросы:

1. Файл в C — это поток байтов, с которым программа работает через указатель FILE*.
2. FILE* — это указатель на структуру, которая описывает файловый поток.
3. Режимы: r (чтение), w (запись), a (дозапись), rb/wb/ab (бинарные), r+/w+/a+ (чтение+запись).
4. Текстовый — хранит символы, бинарный — любые байты.
5. putc записывает символ в файл, getc читает символ из файла.
6. fputs записывает строку, fgets читает строку (с ограничением длины).
7. fprintf форматированная запись в файл, fscanf форматированное чтение из файла.
8. fwrite записывает блок данных в бинарный файл, fread читает блок.
9. fseek перемещает указатель в файле, ftell возвращает текущую позицию.
10. feof проверяет конец файла, ferror проверяет ошибки ввода-вывода.
11. Чтобы сохранить данные, освободить ресурсы и избежать потери/повреждения данных.
