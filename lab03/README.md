### Контрольные вопросы

1. **Что делает malloc?**  
   Выделяет блок памяти заданного размера в байтах. Память не инициализируется.

2. **Чем отличается malloc от calloc?**  
   calloc выделяет память под массив из count элементов по size байт и обнуляет её. malloc — нет.

3. **Что делает realloc?**  
   Изменяет размер ранее выделенного блока памяти. Может переместить блок.

4. **Что произойдет при утечке памяти?**  
   Программа будет занимать всё больше памяти, пока не закончится или не упадёт.

5. **Зачем нужен free?**  
   Освобождает динамическую память, чтобы её можно было использовать снова.

6. **Что произойдет при двойном освобождении памяти?**  
   Ошибка времени выполнения (undefined behavior), программа может упасть.

7. **Как проверить, что malloc вернул NULL?**  
   `if (ptr == NULL) { /* обработка ошибки */ }`

8. **В чем отличие статического и динамического массива?**  
   Статический имеет фиксированный размер, выделяется на стеке, освобождается автоматически. Динамический — размер задаётся в рантайме, память в куче, нужен free.
