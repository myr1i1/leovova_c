# Ответы на вопросы

## Как объявить структуру в языке C?

struct Student {
    char name[100];
    int age;
    float score;
};

## Что такое вложенные структуры и зачем они нужны?

Вложенные структуры — это когда одна структура содержит другую как поле. Нужны для группировки связанных данных, например:

struct Person {
    char name[100];
    struct Date birth_date;  // вложенная структура
};

## Как передеать структуру в функцию?

По значению: void func(struct Student s) — создается копия

По указателю: void func(struct Student *s) — передается адрес (быстрее, можно изменять оригинал)

## Можно ли создавать массивы структур?

Да, например

struct Student group[30];  // массив из 30 студентов

## Как обратиться к элементу массива структур?

group[0].name = "Елизавета";     // через точку
group[0].age = 20;

## Можно ли использовать enum внутри структуры?

Да

enum Course { FIRST, SECOND };
struct Student {
    enum Course course;  // поле типа enum
};

## Можно ли сравнивать структуры оператором ==?

Нет, нельзя. Нужно сравнивать каждое поле отдельно.

if(s1.age == s2.age && strcmp(s1.name, s2.name) == 0) {
    // структуры равны
}
